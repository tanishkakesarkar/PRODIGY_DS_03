# PRODIGY_DS_03

## Task 03 - Decision Tree Classifier

Built a Decision Tree Classifier to predict whether a customer will purchase a product/service using the Bank Marketing Dataset.

## Tasks Performed

* Loaded and explored the dataset
* Converted categorical data into numerical values
* Split dataset into training and testing sets
* Trained a Decision Tree Classifier
* Evaluated model accuracy

## Tools & Technologies Used

* Python
* Pandas
* Scikit-learn
* Google Colab

## Result

The model achieved approximately 88% accuracy.
